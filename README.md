## How to Run

1. Make sure Python 3 is installed.
2. Open a terminal or command prompt.
3. Run the program:


4. Enter:
   - n (between 3 and 11)
   - k (>10)

The program will:
- Search all values of x and y from 10 to k
- Find near misses where x^n + y^n is close to z^n
- Print every new best near miss
- Finally show the **smallest miss** at the end
